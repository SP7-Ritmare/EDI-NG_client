# Datatypes available for EDI Templates
## code (alias: codelist)
Item's value comes from a SPARQL codelist shown as a combo box (HTML **select** element):
returned fields MUST be:
 ?c (uri),
 ?urn (optional urn),
 ?l (in-language label),
 ?a (alternative language label)
 ?z (language-neutral label)
